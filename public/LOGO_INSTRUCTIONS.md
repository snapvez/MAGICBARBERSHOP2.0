# Instruções para o Logo

Para que o logo apareça corretamente na aplicação:

1. Coloque a imagem do logo nesta pasta (`/public`)
2. Renomeie o arquivo para: `logo.jpeg`
3. Formatos suportados: JPEG, JPG, PNG
4. Tamanho recomendado: 200x200 pixels (mínimo)

## Fallback

Se o logo não carregar, a aplicação mostrará automaticamente um ícone de tesoura dourado como alternativa.

## Caminho atual esperado

- `/public/logo.jpeg`

## Como testar

1. Coloque a imagem em `/public/logo.jpeg`
2. Recarregue a página
3. O logo deve aparecer no cabeçalho
